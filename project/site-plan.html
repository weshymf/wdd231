Site Plan: Gaming Highlights

Site Name

Gaming Highlights

This name reflects the site's focus on showcasing and analyzing the top-rated games from the last three years based on Metacritic scores.

Site Purpose

The purpose of this website is to provide an in-depth look at the top 5 games of each of the last three years. The site will include details on game development, critical reception, and why each title stands out.

Scenarios

  • What are the best-rated games from the last three years?
  • Which game studios and engines were used to develop these top games?
  • On which platforms were these games released?
  • What were the key innovations and features that made these games stand out?

Color Schema

  • Primary Color: #9a80e9 (Used for headers, buttons, and accents)
  • Secondary Color: #140c2c (Used for body text and backgrounds)

Typography

  • Heading Font: 'Helvetica Neue', Helvetica, Arial, sans-serif
  • Body Font: Arial, sans-serif
