About Me & The Project

Introduction

Hello! I'm Wesley Ferreira, a passionate developer and data analyst based in Brazil. I created this project to merge two of my favorite interests: gaming and web development.

Why Gaming?

Gaming has always been a major part of my life. With constant innovations in storytelling, visuals, and mechanics, I wanted to spotlight the most impactful games of the past three years according to Metacritic. This website is my way of honoring that creativity while showcasing my development skills.

Development Approach

This site was built from scratch using HTML5, CSS3, and JavaScript. I focused on accessibility, responsiveness, and dynamic data integration to create a smooth user experience across all devices.

About the Project

The project features three detailed pages—one for each year—highlighting five standout games. Each entry includes development information, release details, critical reviews, and the reason for selection. It also includes dynamic JavaScript elements like modals, localStorage, and content built from JSON data.
